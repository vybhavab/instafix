It now also supports:

- Aspect-ratio padding/cropping
- Anamorphic de-squeeze
- Lens-profile-based distortion correction for the Sirui 24mm f/2.8 Anamorphic 1.33x (Fuji X)

The first lens profile is intentionally lightweight and adjustable. It defaults to a mild barrel correction based on published Fuji X user reports that commonly needed roughly `+7` to `+13` distortion correction in Photoshop for this lens.
